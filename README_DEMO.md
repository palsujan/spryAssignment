# Spry Shop (Demo)

This demo app shows products fetched from `https://fakestoreapi.com/products` and demonstrates filtering, sorting, and favoriting.

Run locally:

```powershell
cd d:\React\spryAssignment\spryshop
npm install
npm run dev
```

Open the URL shown by Vite (e.g. `http://localhost:5174/`).

If you want a production build:

```powershell
npm run build
npm run preview
```
